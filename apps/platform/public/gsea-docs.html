Gene Set Enrichment Analysis (GSEA)

Open Targets Platform · Methodology & User Guide

What is GSEA?

Gene Set Enrichment Analysis (GSEA) is a computational method that determines whether a predefined group of genes — representing a biological pathway or function — shows a statistically significant, coordinated pattern within a ranked list of genes. Rather than looking at individual genes in isolation, GSEA asks: do the genes involved in a particular biological process cluster near the top of my disease association list more than expected by chance?

The Platform implements GSEA to reveal biological themes underlying disease associations and support target identification and validation.

Key Terms

GSEA — Gene Set Enrichment Analysis
A method to identify whether predefined groups of genes are overrepresented in a ranked gene list.
Gene set
A predefined collection of genes grouped by shared biological function, pathway membership, or other annotation.
ES — Enrichment Score
A running-sum statistic reflecting how much a gene set is concentrated at the top or bottom of a ranked list.
NES — Normalised Enrichment Score
The ES adjusted for gene set size, allowing fair comparison across gene sets of different sizes.
FDR — False Discovery Rate
A multiple testing correction that estimates the proportion of results that are false positives.
Leading edge genes
The subset of genes within a gene set that contribute most to the enrichment signal and are the strongest candidates for follow-up.

Input Gene Lists

Gene lists for GSEA can be derived from the disease association page using different evidence types:

  • All disease associations — comprehensive ranked list across all evidence types
  • Specific data type or source — subset based on individual data types or sources
  • Custom gene list — user-provided genes with disease associations

Recommended gene list size: The Platform recommends using gene lists of >500 genes to achieve statistically significant results, though the optimal size depends on the gene set library being tested. Larger gene lists generally provide better statistical power and increase the likelihood of detecting meaningful enrichments.

GSEA Implementation

The Platform uses blitzGSEA to perform enrichment analysis. This implementation estimates p-values by fitting enrichment score null distributions to gamma models[1], enabling efficient computation across large gene set collections.

Since all genes from the Platform have positive ranking scores, the analysis is performed one-sided by default. Results are filtered by NES to show only gene sets enriched at the top of the ranked list (NES > 0).

By default, the analysis uses all gene sets from the selected library as the background for enrichment testing. This approach ensures consistent statistical evaluation across all pathways and functional categories within each gene set collection.

Interpreting Enrichment Results

How GSEA Works (Conceptually)

GSEA walks down your ranked gene list from top to bottom. For each gene set, it keeps a running tally: the score increases each time a gene belonging to the set is encountered, and decreases when a gene outside the set is encountered. The maximum deviation of this running sum from zero is the Enrichment Score (ES). If genes in the set cluster near the top of the list, the ES will be high.

Key Metrics

Normalised Enrichment Score (NES)

The NES is the primary metric for interpreting results. It is the ES adjusted for gene set size, which allows you to compare results across gene sets of different sizes.

  • NES > 0: the gene set is enriched at the top of the ranked list (i.e. among the most strongly disease-associated genes)
  • Higher NES values indicate stronger enrichment signals
  • NES > 1.5 is a commonly used threshold for meaningful enrichment, though context matters[4]

p-value

The p-value indicates the statistical significance of the enrichment score, estimated through blitzGSEA's gamma distribution approximation. A low p-value (typically < 0.05) suggests the observed enrichment is unlikely to have occurred by chance.

FDR (False Discovery Rate)

When many gene sets are tested simultaneously, some will appear enriched by chance alone. The FDR corrects for this multiple testing problem. An FDR < 0.25 is the conventional threshold recommended for GSEA results, which is more lenient than the typical 0.05 threshold used in single-hypothesis testing because GSEA is often used as a hypothesis-generating tool[4].

Leading Edge Genes

The leading edge subset represents the core genes driving the enrichment signal. These are the genes that appear at the top of the ranked list and before the point where the enrichment score reaches its maximum. They are the most biologically relevant genes within the enriched gene set and are strong candidates for follow-up investigation.

Quick Interpretation Guide

NES FDR Interpretation
> 2.0 < 0.05 Strong enrichment — high confidence that this pathway is relevant
1.5 – 2.0 < 0.25 Moderate enrichment — worth investigating further
1.0 – 1.5 > 0.25 Weak enrichment — treat with caution, may not be meaningful
< 1.0 any Not enriched — gene set is not concentrated at the top of the ranked list

These thresholds are guidelines, not rigid cutoffs. Biological context and the number of gene sets tested should inform interpretation.

Result Views

The Platform provides three ways to explore GSEA results, each suited to different stages of analysis.

View Description Best used for
Table View Sortable, filterable table listing all enriched gene sets with NES, p-value, and FDR. Scanning results numerically, sorting by significance or enrichment strength, identifying top-ranked gene sets.
Tree View Hierarchical display showing how enriched gene sets relate to each other within the library's organisational structure. Understanding broader biological context — e.g. whether multiple enriched pathways share a top-level process.
Sunburst View Radial, multi-level chart where inner rings represent broad categories and outer rings represent specific gene sets. At-a-glance overview of dominant biological areas; spotting clusters of related enrichments; visual communication of findings.
