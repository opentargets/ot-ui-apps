# Open Targets Partner Preview Platform

> The Open Targets Partner Preview Platform (PPP) is the partner-private instance of the Open Targets Platform. Alongside all public data it exposes partner-private datasets and widgets. This file orients AI agents to the Platform's resources and APIs.

The Platform is organised around entity pages — target, disease, drug, variant, study and credible set — each aggregating evidence and associations for that entity. Pages live under `https://partner-platform.opentargets.org/<entity>/<id>` (for example `/target/ENSG00000169174`). Programmatic access is via the GraphQL API listed below. This partner instance additionally surfaces partner-private datasets and widgets.
